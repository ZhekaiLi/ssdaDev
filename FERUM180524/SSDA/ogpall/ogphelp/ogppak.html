ogppak

Purpose

Puts the Sparse OGP hyperparametrs into a vector.

Synopsis 

hp = ogppak;

Description

hp = ogppak takes the GLOBAL Gaussian Process data structure net and combines the hyperparameters into a single row vector hp. The hyperparameter vector is needed to optimise the OGP structure.

The transformed hyperparameters returned by this function are used in the MLII optimisation method.
