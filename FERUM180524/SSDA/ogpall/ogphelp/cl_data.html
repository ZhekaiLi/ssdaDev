cl_data

Purpose

Generates two-dimensional dataset for classification.

Description

[x,y] = cl_data(N,fname) - generates data from a mixture of 3 Gaussians and labels them with +/-1 where the first class has two components and the second class is a single Gaussian.

[x,y] = cl_data(N,fname,row) - if row is nonzero, then the generated inputs are on a grid with the size N (thus the number of examples is NxN) and the corresponding inputs are -1,1.
