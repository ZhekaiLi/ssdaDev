ogpcovdiag

Purpose

Calculates the diagonal of the covariance function for the OGP.

Synopsis 

covf = ogpcovdiag(x1);

Description

[covf] = ogpcovdiag(net, x1) takes an OGP data structure net together with a data matrix x1 and computes the diagonal elements of the covariance function.

If the output is not one-dimensional, then the returned matrix is [dim(x1),nout,nout] dimensional.

See Also

ogp, ogpinit, ogpcovarf, ogpcovarp
