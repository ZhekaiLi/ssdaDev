demogp_reg_gui

Purpose

Graphical frontend to Online Gaussian Process regression.

Description

demogp_reg_gui - provides a graphical user interface for Online Gaussian Process regression.

This demo illustrates the flexibility of the Online Gaussian Processes for regression using additive noise with Gaussian, Laplace, or positive-exponential distributions. All OGP operations can be reached using the control buttons on the left of the window.

One can set (or reset) the model hyperparameters (such as kernel type or kernel parameter) but in this case the GP inference is restarted.

This program is designed for a first contact with GP and the idea of sparsity within a full probabilistic inference.
